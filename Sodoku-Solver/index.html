Welcome to my Sodoku Solver!

Here's how this works:

This Solver takes two kinds of inputs: either the numbers are entered in manually
or a file in inputted to the Solver. If entering in manually, use the drop downs
to select a number for the desired space. Of course, for a valid Sodoku, you must
bear in mind these rules:

  1. No duplicate numbers in a row
  2. No duplicate numbers in a column
  3. No duplicate numbers in a box
  4. For a valid Sodoku puzzle, there must be at least 17 filled spaces to start

Well, that's the long and short of it. Good luck!
